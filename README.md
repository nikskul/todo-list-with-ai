# TODO List — Веб-приложение

Простое веб-приложение для управления задачами, построенное на **Vue 3 + TypeScript + Vite**.

## Возможности

- ✅ Создание задач с заголовком и описанием
- ✅ Просмотр списка активных задач
- ✅ Отметка задач как выполненных
- ✅ Редактирование заголовка и описания
- ✅ Архивирование и восстановление задач
- ✅ Данные сохраняются в localStorage браузера

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build
```

Приложение откроется по адресу `http://localhost:5173`.

## Доступные команды

| Команда | Описание |
|---------|----------|
| `npm run dev` | Запуск dev-сервера с HMR |
| `npm run build` | Продакшен-сборка |
| `npm run preview` | Предпросмотр продакшен-сборки |
| `npm run test` | Запуск всех тестов (watch mode) |
| `npm run test:run` | Запуск всех тестов (один раз) |
| `npm run lint` | Проверка линтером |
| `npm run format` | Форматирование кода |

## Архитектура

```
src/
├── types/           # TypeScript типы и интерфейсы
├── services/        # Сервисный слой (работа с localStorage)
├── composables/     # Composition API хуки (бизнес-логика)
├── components/      # Vue компоненты
├── App.vue          # Корневой компонент
└── main.ts          # Точка входа
```

## Тестирование

```bash
# Запустить все тесты
npm run test

# Запустить с покрытием
npm run test:run -- --coverage
```

## Стек технологий

- **Vue 3** — фреймворк с Composition API
- **TypeScript** — строгая типизация
- **Vite** — быстрый сборщик
- **Vitest** — тестирование
- **localStorage** — хранение данных в браузере
